## 実行

ルートディレクトリ上で下記のコマンド

```
gradle run
```

### テスト実行

```
gradle test
```

### javaDoc生成
```
gradle javadoc
```

### jarファイル生成

生成
```
gradle jar
```

実行
```
java -jar app/build/libs/app.jar
```
